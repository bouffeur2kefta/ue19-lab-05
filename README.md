# UE19 Lab 05 - Random Joke API

Application Python qui récupère et affiche des blagues aléatoires en utilisant l'API JokesAPI.

## Fonctionnalités

- Récupération d'une blague aléatoire depuis https://official-joke-api.appspot.com
- Affichage formaté de la question et de la chute
- Gestion des erreurs réseau et de parsing

## Installation

### Prérequis
- Python 3.11+
- pip
